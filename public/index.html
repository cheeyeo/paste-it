My-Snippets is...


A code highlighting site for users to post 'snippets' of code or text. The core concept owes a lot to well established code-highlighting sites such as Pastie. However the idea is to provide an alternative service to accomplish the same goals.


Current features:

  • Support for 30+ file formats and languages

  • Ability to edit snippets after posting (valid for up to an hour)

  • Permalinks for posted snippets and downloads

  • Categorization of file extensions

  • Fast and easy to use (uses Pygmentizer to process code snippets)


Future features:

  • Ability to upload entire source files

  • Commit to remote repository using GIT

  • Ability for users to store and edit personal snippets

  • Support for more languages

  • Smarter searching
